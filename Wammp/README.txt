In order to make WAMMP work, the following files are needed:

Bass.Net.dll	http://www.un4seen.com/download.php?z/4/Bass24.Net.zip
bass.dll		http://www.un4seen.com/download.php?bass24
bassmix.dll		http://www.un4seen.com/download.php?bassmix24

The files above must be placed on same directory of wammp.exe
